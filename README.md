# awayflayer.github.io

This repository hosts the source code for the `awayflayer.github.io` website, a JavaScript application.

## Description

`awayflayer.github.io` is a web application built primarily with JavaScript, CSS, and HTML. This website serves as a personal or project page, showcasing various projects, blogs, or any other content.

### Sub-Applications

1. **Timer Application**:
    - A simple timer application with functionalities to start, stop, clear, export, and import timers.
    - Timers are dynamically injected into the main content area.

2. **Wheel Application**:
    - A spinning wheel application for selecting random options.
    - Features include adding options, spinning the wheel, clearing options, and exporting/importing wheel configurations.

3. **Statistics Analyzer**:
    - An application for analyzing statistics from JSON files.
    - Supports visualization with pie and bar charts, zooming, panning, and data summaries.

## Getting Started

To get started with the project, follow these steps:

1. **Clone the repository**:
    ```
    git clone https://github.com/AwayFlayer/awayflayer.github.io.git
    ```
2. **Navigate to the project directory**:
    ```
    cd awayflayer.github.io
    ```
3. **Install dependencies** (if any):
    ```
    npm install
    ```
4. **Run the application**:
    ```
    npm start
    ```
